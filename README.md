**3. Tính năng phần mềm** 
   a, Tính năng: Nhập dữ liệu đầu vào
       - Nhập kích thước cột (b, h, l).
       - Nhập thông số vật liệu (Rb, Rs, Rsc).
       - Nhập nội lực tác dụng (N, Mx, My).
       - Nhập điều kiện liên kết (ψ).
       - Kiểm tra tính hợp lệ dữ liệu (giá trị không âm, không rỗng, đúng theo TCVN, không chia cho 0, cảnh báo khi nội lực quá nhỏ → xem xét là nén đúng tâm).
   b, Tính năng: Phân loại cột
       - Nén đúng tâm: Mx = 0, My = 0.
       - Nén lệch tâm một phương: Mx ≠ 0 hoặc My ≠ 0.
       - Nén lệch tâm xiên: Mx ≠ 0 và My ≠ 0.
   c, Tính năng: Tính toán cơ bản
       - Chiều dài tính toán: l0 = l × ψ.
       - Bán kính quán tính: i = √(I/Ab).
       - Độ mảnh λ = l0 / i.
       - Xác định hệ số giảm khả năng chịu lực φ.
   d, Tính năng: Tính toán khả năng chịu lực
       - Lực nén thiết kế Nsd.
       - Mô men thiết kế theo trục X: Mx_thiết_kế.
       - Mô men thiết kế theo trục Y: My_thiết_kế.
       - Tính diện tích cốt thép dọc cần thiết: As_tính_toán.
   e, Tính năng: Bố trí và kiểm tra cốt thép
       - Kiểm tra diện tích cốt thép: As_min ≤ As ≤ As_max.
       - Bố trí sơ đồ thép dọc (4, 6, 8 thanh…).
       - Kiểm tra cấu tạo (khoảng cách, lớp bảo vệ).
   f, Tính năng: Hiển thị kết quả
       - Xuất bảng kết quả tính toán.
       - Hiển thị hình minh họa tiết diện bố trí thép 2D.
       - Thông báo cảnh báo nếu không đạt yêu cầu.
